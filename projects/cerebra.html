Cerebra

Student wellness and productivity app

A wellness platform built to help students manage their mental load alongside academic responsibilities. Cerebra combines productivity tools with wellness features and a push notification system that required deep backend debugging to get production-ready.

Role Full-stack developer
Stack Supabase, Edge Functions
Type Client project
Status Live
The build

Cerebra is designed around a simple but important idea: students carry two parallel loads: academic pressure and mental load. And most tools only address one. The platform brings both into one place: study streak tracking, journaling, quick access to wellness resources, and timely push notifications to keep users engaged without overwhelming them.

The backend runs entirely on Supabase Edge Functions — serverless, deployed at the edge, with no traditional server to maintain.

Key features

Study streak tracking

Daily check-in calendar with streak visualization to keep students engaged and accountable over time.

Push notification system

Scheduled, serverless push notifications via Supabase Edge Functions — keeping users engaged without requiring them to be active in the app.

Wellness and productivity tools

Tasks, journaling, free e-resources, and document upload — all accessible from a single clean dashboard.

Serverless backend

Entire backend runs on Supabase Edge Functions — no server to provision, maintain, or scale manually.
